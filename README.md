**To-Do Webpage**

A simple, interactive To-Do List web application built using HTML, CSS, and JavaScript. This project allows users to manage their daily tasks efficiently with features like adding, deleting, and marking tasks as completed.

**Features**

Add Tasks: Easily add new tasks to your list.
Delete Tasks: Remove tasks that are no longer needed.  

**Technologies Used**

HTML5 – Structure of the webpage.
CSS3 – Styling and responsive design.
JavaScript – Dynamic task management.
